1. 下载 anaconda
https://www.anaconda.com/

2.  打开控制台 打开Anaconda的Prompt
输入【conda env list】可查看当前所有已创建好的环境

3.  创建环境
  conda create -n Fuel python=3.10

4. 切换虚拟环境的命令格式
conda activate 虚拟环境名字

5. 安装库
pip install -U xbx -i https://pypi.tuna.tsinghua.edu.cn/simple
